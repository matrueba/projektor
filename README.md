# projektor
Multi-scene Video Generator using IA

# Requirements
* Docker
* For cloud execution: ComfyUI with API enabled andCloud Platform (Runpod recommended)
    - Explained in section "Cloud Execution"
* For local execution: ComfyUI with API enabled
    - Explained in section "Local Execution"

# Installation
sudo docker run -it --rm --name projektor -p 3000:3000 projektor
